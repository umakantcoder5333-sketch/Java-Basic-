# Image Gallery Project

This project I made during my internship at CodeAlpha.

In this project, I created a simple image gallery using HTML, CSS and JavaScript.

## What I did in this project
- Created a gallery layout using CSS grid
- Added different categories like nature, water, mountain, bird, flower
- Made filter buttons to show images based on category
- Added lightbox so when we click image it opens in full screen
- Added next and previous buttons to navigate images
- Added hover effect and smooth transition
- Made it responsive for different screen sizes

## How it works
- Click on any image to open it
- Use next/prev buttons to move between images
- Use buttons to filter images

## Tools used
- HTML
- CSS
- JavaScript
